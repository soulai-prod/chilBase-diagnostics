## AI-зрелость: предварительная оценка (до интервью)

| Отдел | AI-зрелость | Обоснование |
|-------|------------|-------------|
| Product | 🟢 HIGH | Chill-AI, ML LiveOps, активно используют Claude/GPT/Gemini |
| Marketing (Performance) | 🟢 HIGH | AI creative pipeline (Claude Code, Veo 3, Nano Banana) |
| Media Marketing | 🟡 MEDIUM | Эксперименты с AI-баннерами, Claude Code + MCP |
| Meta Product | 🟡 MEDIUM | Все 3 модели активно используются |
| Community | 🟡 MEDIUM | ChatGPT для опросов, AppFollow AI, хотят Helpshift AI |
| QA | 🟡 LOW-MED | ChatGPT/Gemini/Claude используются, смотрят TestRail AI |
| Environment | 🟡 LOW-MED | Точечно: ChatGPT, Gemini, Freepik — нет production pipeline |
| Finance | 🟡 MEDIUM | ChatGPT/Claude для анализа, Excel AI, Tableau AI |
| Legal | 🔴 LOW | Только ChatGPT для переводов и простых документов |
| HR | 🟡 LOW-MED | ChatGPT/Claude/Gemini личные, Skailer для скрининга |
| Bizdev | 🔴 LOW | ChatGPT/Gemini для ресерча, нет системного подхода |
| Production | 🟡 LOW-MED | Copilot + ChatGPT, нет системности |
| 2D | ❓ UNKNOWN | Данных нет, нужно выяснить |
