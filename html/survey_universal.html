AI-грамотность

Здесь нет «правильных» ответов с карьерными последствиями. Вопросы нужны, чтобы правильно подобрать формат обучения.

Профиль AI-зрелости
1Что такое LLM (Large Language Model)?
2Что такое «галлюцинация» LLM?
3Мини-задача: выберите лучший промт
Ситуация

Вам нужно попросить ChatGPT написать короткое саммари встречи: 5 ключевых решений, кто ответственный, дедлайны. У вас есть транскрипт на 3000 слов.

Какой промт даст наиболее полезный результат?
A
«Сделай саммари»
B
«Напиши саммари встречи из этого текста: [транскрипт]»
C
«Проанализируй транскрипт встречи и выдай: 1) Список ровно 5 ключевых решений; 2) Для каждого — ответственный (если упоминался) и дедлайн (если назывался). Формат: нумерованный список. Если дедлайн не назывался — пиши "не определён". [транскрипт]»
D
«Что было на встрече?»
Правильный ответ — C. Чёткая структура, конкретный формат вывода и инструкция для крайних случаев дают предсказуемый результат. Без этих деталей LLM выдаст расплывчатое резюме.
