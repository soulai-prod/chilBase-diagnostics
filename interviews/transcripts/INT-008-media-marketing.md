### Боли

**Визуальные материалы:**
- Зависимость от арт-отдела (рендеры)
- Лица/персонажи в AI — не удаётся сохранить vibe, качество модельки мобильной игры недостаточно
- На каждый арт персонажи делаются "на несколько тонов лучше чем в игре"
- Видеоконтент через AI — "через раз получается", прогрессирует с каждой моделькой

**Локализация — огромная боль:**
- Работают по всему миру, обширная география
- Отдел локализации с кучей переводчиков, неделя на перевод
- Хочет: не перевод EN→другие языки, а генерация контента на каждом языке из идеи поста
- Тестировали LLM для переводов — качество растёт, разные модели лучше для разных языков
- Не видят локальные тренды (Latam, Япония) — только русскоязычные

**Тренд-вотчинг:**
- Нет систематизированного процесса
- Brand Mentions, Sprout Social — ручная работа, пролистывание
- Хочет: бот в Slack с утренним дайджестом трендов, конкурентов, новостей индустрии

**SMM posting:**
- Много аккаунтов, нет единого инструмента для постинга в несколько аккаунтов
- Ручное копирование текстов, картинок в каждый аккаунт
- Discord тоже требует ручной постинг
- Ручной ответ на комменты — не автоматизирован

**Инфлюенсеры:**
- Документы и оплаты — полностью ручной труд

- Нет атрибуции видео → инфлюенсер → дашборд
- Отсмотр контента инфлюенсеров — по 1.5 часа на видео, проверка бренда, логотипов

**Project management:**

- Хочет: план → задачки → статусы → фильтрация по проекту/коллаборации
